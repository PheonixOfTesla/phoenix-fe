===============================================================================

ENTRY POINTS:
  ✓ Web: index.html (login) → onboarding.html → dashboard.html
  ✓ iOS: Capacitor build from webDir

MAIN FEATURES:
  ✓ Authentication: JWT-based with token refresh
  ✓ Voice Interface: Wake word + continuous mode
  ✓ 7 Planetary Systems: Health, fitness, calendar, goals, finance, legacy, custom
  ✓ Wearables: 5 device types supported
  ✓ Multilingual: 37+ languages
  ✓ Real-time: Caching, pub/sub state updates

KEY FILES TO MODIFY:
  Config changes: src/config.js
  API changes: src/api.js (or backend)
  UI changes: src/*-app.js or *.html
  Styling: styles.css, phoenix-*.css
  Voice: src/voice-*.js
  State: src/planets.js

TESTING STARTING POINTS:
  npm test (runs test-full-flow.js)
  npm run dev (starts local server)
  Individual test: node test-dashboard.js
